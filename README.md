This portfolio showcases my learning journey in web development.
Each task demonstrates a specific concept — from basic HTML structure to advanced UI interactions.

This project is a complete front-end portfolio created as part of a web programming course.
It includes 50 structured tasks covering HTML, CSS and JavaScript concepts.

The website includes:

A Home page introducing the project
A Tasks page with all 50 tasks and live demos
A Contact page

The 50 tasks are grouped into the following sections:

HTML Foundations (Tasks 3–17)
Basic structure, lists, tables, layouts
CSS & Layouts (Tasks 18–27)
Flexbox, forms, styling, responsive design
JavaScript (Tasks 28–41)
DOM manipulation, events, validation
Advanced (Tasks 42–45)
Z-index, layering, interactive UI elements
